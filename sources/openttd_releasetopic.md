[b]Extras:[/b]
[list]
[*]Detailed [url=https://raw.githubusercontent.com/harbonah/games-openttd-scenario-hungary/refs/heads/main/hungary_8Kx4K_scaled_down_hungarian_town_names.json]town list[/url] featuring nearly 8,000 settlements (around 200 designated as cities). Populations have been scaled down to enable earlier start dates, so the three capital cities (Budapest, Bratislava, Wien) are slightly smaller.
[/list]

Before starting, load the selected scenario in the scenario editor, configure your Game Script and start date, and select the NewGRFs you wish to use.
